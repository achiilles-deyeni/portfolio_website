Some Projects

Loan Default Prediction Model

Developed a machine learning model using Python to predict loan defaults based on historical data. The model achieved an accuracy of 85%. This project involved data preprocessing, feature engineering, and model evaluation using libraries such as Pandas, Scikit-learn, and Matplotlib. The model can be used by financial institutions to assess the risk of loan applicants and make informed lending decisions.

easeGroupings

easeGroupings is a desktop application built using JavaFX and Scenebuilder. This application takes a list of people in .txt or .xlsx format and put them into groups randomly based on the number of people needed to be in a group. This project was built to help ease the stress of groupings in schools.

Stock Prediction Model

Build a stock price prediction model using Python and machine learning techniques. The model utilizes historical stock data and technical indicators to predict future price movements. The project involves data collection, preprocessing, feature selection, and model training using libraries such as Pandas, NumPy, and Scikit-learn. The model can assist investors in making informed buy or sell decisions based on predicted stock trends.

Weather Prediction Model

Built a weather prediction model using Python and machine learning algorithms. The model utilizes historical weather data and various features such as temperature, humidity, and wind speed to make accurate predictions. The project involves data collection, preprocessing, feature selection, and model training using libraries such as Pandas, NumPy, and Scikit-learn. The model can be used by meteorologists and researchers to improve weather forecasting accuracy. I used FB Prophet for this project.

ABRATO

Built a fully functional backend for a chemical shop. This was done for an academic purpose which utilized tools and frameworks which include: MongoDB, Node.js and Express.js. JWT authentication was used for authentication of users. Postman was used in testing the routes. The project was a group project and it further solidified my skills in git/github, postman, project management and JWT authentication.
